Microsoft Entra 
	: Known as Azure AD which can be used to manage user and devices for a company.
	: Microsoft Entra is used to assign specific roles to user. this is known as role based access control (RBAC).
	: Several different roles can be assigned to users in the organization like global administrator, global reader
